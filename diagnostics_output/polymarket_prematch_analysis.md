# Polymarket Pre-Match Analysis — June 16, 2026

## Methodology
1. Get Polymarket prices for upcoming matches
2. Get ESPN/DraftKings betting odds for the same matches
3. Convert both to implied probabilities
4. Calculate the spread/divergence
5. Only recommend markets where:
   - Polymarket has existing liquidity (not empty books)
   - Betting odds are available from ESPN
   - The divergence is small enough to be safe
   - We place quotes OUTSIDE the existing spread (wider)
